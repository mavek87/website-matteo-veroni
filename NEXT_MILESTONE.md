## Milestone to achieve

**Complete redesign of matteoveroni.com** — rebuild the site from scratch as a high-impact personal brand platform targeting clients, companies, and CTOs who need elite-level technical services.

---

### Tech Stack

- **Astro** (static site generator) — chosen for native Markdown/Content Collections support (blog) and built-in i18n routing
- **Vanilla HTML/CSS** via CSS Variables — no heavy CSS frameworks

### Aesthetic Direction (Candidate — not yet decided)

A possible direction explored is **Industrial Utilitarian / High-Tech Brutalism**, to communicate precision, technical authority and high seniority. This is NOT a final decision — the graphic style must still be evaluated and approved by Matteo before development begins.

Possible details of this direction (all subject to change):
- Background: Void Black `#050505`, text: Ice White `#F4F4F5`, grid/borders: Dark Slate `#27272A`
- Single accent color (Electric Cyan `#00E5FF` or Terminal Amber `#FFB000`) for hover states and CTAs only
- Layout divided by visible thin grids (like dark graph paper)
- Sections load with micro-animations simulating an OS boot sequence
- Typography: **JetBrains Mono** (monospace) for code/structure elements + **Space Grotesk** for display titles

Four layout prototypes were explored (Raw Terminal, Engineering Blueprint, System Dashboard, Dark Brutalism). These are only the first prototype proposals but not all the possible!!!

---

### Non-Negotiable Requirements

1. **Full i18n** — translated into all existing languages (as on current site)
2. **Perfectly responsive** — mobile, tablet, desktop
3. **Excellent content** — reflects who Matteo is, what he can do, his skills, and drives business
4. **Blog with Markdown support** — `.md` files dropped into a folder, Astro generates pages automatically
