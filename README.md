### Quick start

```bash
pnpm install
pnpm build
```

For development details and architecture, see [AGENTS.md](AGENTS.md).

### Deployment

Pushing to the `main` branch triggers an automatic build and deploy via Netlify.

### Services used in production

- [Namecheap](https://www.namecheap.com/) - domain (www.matteoveroni.com)
- [Netlify](https://www.netlify.com/) - hosting and auto-deploy
